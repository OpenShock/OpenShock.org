# OpenShock Landing Page

Just the Landing Page at [OpenShock.org](https://openshock.org)

Built with SvelteKit 2 and Svelte 5, using the Cloudflare Pages adapter.

## Contributing

How to run the project locally:
- `pnpm install`
- `pnpm dev`

SvelteKit should now be serving at `http://localhost:5173`.
